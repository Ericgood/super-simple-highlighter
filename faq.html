Frequently Asked Questions

I just installed it, and nothing’s happened. What an I supposed to do?

Until a highlight is made on a page, no change will be visible. To highlight text you must select it, then right click on it, and then choose a style to define it by. From now on, an icon will be shown in the URL bar, and clicking it will show the page’s highlights.

The icon is grey!

If the icon is grey (or isn't visible at all), it means there are no records of highlights on the page. If there are records of highlights, but they can't be located on the page, the icon is in full colour but is overlayed with a red sign

The icon (with the blue pen) disappeared!

Chrome sometimes hides the icon of extensions at the top of the main menu. You can reveal it by opening the menu, and if you desire move it back into the main bar by right-clicking on the icon.

Can I see a full page containing just the highlights on the page (i.e an overview/summary page)?

From the popup box that opens when you click the icon in the URL bar, click the button with 3 lines on it (AKA the hamburger menu), and choose Open Overview in New Tab from the menu.

When I reload the page the highlights don’t reappear!

This happens occasionally. There are many ways that a page design is incompatible with how this extension works. The more fancy (i.e. dynamic) it is, the less likely it will work. Try a test highlight first, and if it restores next time, it should be OK. But don't bank on it always working.

How are the highlights stored?

A database is created, associating the page's URL with the structural position on the page of the highlight, its style, and its text. When the page is next reloaded, its URL is checked, and if it matches an existing record, it attempts to restore the highlights in the same position. If any part of this sequence is not possible, the highlights can’t be restored.

Note the highlights are ONLY stored on the computer they're made on. They aren't sent anywhere, or backed up (other than as part of a standard computer backup). So if they're lost there, they're lost everywhere

How do I change the title or colour of highlights, see which pages contain highlights, or remove a page from the database?

Via the Options page. Click the close button next to the page. Pages are listed under the Pages tab.

I added highlights to a file I saved to my computer, but when I reload it they’re not there, and the icon has a red sign over it.

By default, extensions can’t make alterations (e.g. show highlights) for locally stored files. It only works for actual web sites. This is a security measure. To override this, and allow the extension to restore the highlights, go to the Chrome Extensions page, and enable ‘Allow access to file URLs’ for the ‘Super Simple Highlighter’ extension.

How can I make/restore highlights in Incognito mode?

As previously, by default Chrome disables extensions in Incognito mode unless explicitly allowed. You can enable it via the Chrome Extensions page, with the ‘Allow in incognito’ checkbox. Keep in mind that by making a highlight on a page whilst it is in Incognito mode, the URL of the page is stored in your list of highlighted pages. So make sure you’re OK with its URL potentially becoming public.

How do I edit highlight styles, remove the text selection after highlighting, or see which pages I’ve highlighted?

All of these are accessible via the extension’s Options page. You access this page from either the Chrome Extensions page, or, if present, by right clicking on the highlighter icon in the url bar, and choosing Options.

How do I assign a keyboard shortcut to the act of creating a highlight?

Go to the Chrome Extensions page, and at the very bottom of the page is a Keyboard shortcuts link, which manages the shortcuts for all extensions. You can assign a shortcut to any of the first 10 highlight styles you may have defined. There is a limit of 10 shortcuts because Chrome doesn't allow me to allocate or remove then as you go. I have to reserve them upfront.

Why are my highlights not synced between my computers?

Chrome offers automatic sync services for free, but only for small amounts of data, such as extension settings. Super Simple Highlighter uses this to sync highlight styles along with its other minor settings. If you define a new highlight style, it should eventually appear on your other computers. However, the actual highlight database (which stores the URL of highlighted pages and the location on the page of its highlights) is too large to use this service.

The ideal situation would be to use a central server to store highlights, but, frankly, I have no intention of funding, securing, and maintaining that service for free.

Therefore, I can only offer the option to export the database from one computer, and import the file onto another, like we're living in the dark ages again. This is in the Advanced tab of the extension's Options page.

You can find this function in the Advanced section of the options page.

How do I access the Chrome Extensions page?

Via any one of these methods:

  • On the main menu (at the top right of the browser), in the More Tools section.
  • If you use a Mac, it’s on the Windows menu.
  • Enter chrome://extensions in the URL bar.
  • If the highlighter icon is present in the URL bar, right click on it and choose Manage.

Can I save the web page to a file, with the highlights on it?

Yes (as of version 3.8.3), but you need to use Web page, Complete format in the Save dialog box, NOT Web page, HTML only nor Web page, Single File (which shouldn't even be enabled by default).
